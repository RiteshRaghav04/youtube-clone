# YouTube Clone

A responsive YouTube-inspired web application built using React, TypeScript, Vite, and Tailwind CSS.  
This project recreates the modern YouTube UI with a clean design, responsive layout, and reusable components.

## Features

- Responsive YouTube-style homepage
- Video cards and thumbnails
- Sidebar navigation menu
- Search bar UI
- Mobile-friendly layout
- Built with reusable React components
- Fast development using Vite

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- CSS3

## Folder Structure

src/
 ├── components/
 ├── data/
 ├── App.tsx
 ├── main.tsx
 └── index.css

## Installation

Clone the repository:

git clone https://github.com/your-username/youtube-clone.git

Go to project folder:

cd youtube-clone

Install dependencies:

npm install

Start development server:

npm run dev


## Build for Production

npm run build

## Preview Production Build

npm run preview
